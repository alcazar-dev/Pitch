Análisis Multivariado de Contaminantes en el Aire

Descripción
Proyecto de análisis ambiental que integra técnicas geoespaciales, multivariadas y de aprendizaje automático para estudiar patrones de contaminación atmosférica. Incluye visualizaciones interactivas, modelado predictivo y análisis de clusters.
Competencias Técnicas
  • Programación avanzada en Python con Pandas, NumPy, Matplotlib
  • Análisis geoespacial con GeoPandas (WGS84 EPSG:4326, Web Mercator EPSG:3857)
  • Visualizaciones interactivas con Contextily
  • Automatización de pipelines de datos
Competencias Matemáticas/Estadísticas
  • Análisis multivariado (PCA, Análisis Factorial)
  • Pruebas estadísticas (KMO, Bartlett)
  • Clustering (K-Means) con método del codo
  • Correlación multivariada
Desarrollo de Modelos
  • Redes LSTM multicapa para predicción temporal
  • Interpolación espacial (griddata cúbico)
  • Validación con cross-validation (MSE, R², MAE)
  • Arquitecturas para imputación de valores faltantes

Restauración Ecológica del Altiplano Mexicano

Descripción
Solución de optimización combinatoria para el problema de ruteo de vehículos (VRP) aplicado a proyectos de restauración ecológica. Combina métodos exactos y heurísticos para manejar restricciones complejas de capacidad, tiempo y costos.
Competencias Técnicas
  • Modelado algebraico avanzado en GAMS
  • Algoritmos heurísticos en Python
  • Estructuras de datos eficientes para matrices de distancia
  • Interfaces de traducción modelo-implementación
Competencias Matemáticas
  • Programación entera-mixta (MIP)
  • Formulación Miller-Tucker-Zemlin para subrutas
  • Análisis de complejidad algorítmica
  • Modelado con ecuaciones diferenciales
Soluciones Implementadas
  • Solver exacto en GAMS (gap del 1%)
  • Heurística KNN con refinamiento local
  • Optimización multiobjetivo
  • Sistema de penalizaciones dinámicas
  • Validación cruzada de soluciones

Clasificación de Imágenes con Vision Transformer (ViT)

Descripción
Implementación avanzada de arquitecturas transformer adaptadas a visión por computadora. El modelo procesa imágenes como secuencias de parches, aplicando mecanismos de atención para capturar relaciones espaciales complejas, logrando alta precisión en tareas de clasificación.
Competencias Técnicas
  • Implementación de ViT con TensorFlow/Keras
  • Capas personalizadas para procesamiento de parches
  • Mecanismos de atención multi-cabeza (4 cabezas)
  • Aumento de datos avanzado
  • Optimización de rendimiento
Competencias Matemáticas
  • Multi-Head Attention con proyecciones matriciales
  • Normalización por capas y skip connections
  • Optimización con AdamW (lr=0.001)
  • Función de activación GELU
  • Regularización estratificada (dropout 0.1-0.5)
Ingeniería de Modelo
  • Arquitectura ViT con 8 capas transformer
  • Hiperparámetros optimizados: parche 6x6, proyección 64
  • Redimensionamiento estratégico de imágenes
  • Pipeline con early stopping adaptativo
  • Sistema de visualización avanzada
  • Evaluación exhaustiva con múltiples métricas
